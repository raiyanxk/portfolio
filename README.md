# Raiyan Khan's Personal Portfolio

A personal portfolio website designed to showcase projects, experience, and skills.

**Live:** [raiyan-khan.vercel.app](https://raiyan-khan.vercel.app)

## Technologies

- Next.js 15
- React 19
- Tailwind CSS v4
- Motion for animations

## Getting Started

Clone the repository and install dependencies:

```bash
git clone https://github.com/raiyan/portfolio.git
cd portfolio
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site locally.
